# Logistics Vehicle Scheduling Optimization

## Project Overview
Mathematical modeling project for optimizing logistics vehicle scheduling and routing, developed for a mathematical modeling competition.

## Features
- **Problem 1**: Package volume prediction using machine learning
- **Problem 2**: Multi-objective vehicle scheduling optimization
- **Problem 3**: Container utilization analysis
- **Problem 4**: Robust scheduling under uncertainty

## Project Structure
```
├── src/
│   ├── problem1/     # Prediction models
│   ├── problem2/     # Vehicle scheduling
│   ├── problem3/     # Container analysis
│   └── problem4/     # Robust optimization
└── outputs/          # Results and visualizations
```

## Key Technologies
- Machine Learning (XGBoost, Random Forest)
- Multi-objective Optimization (NSGA-II)
- Linear Programming
- Sensitivity Analysis
- Robust Optimization

## Models Used
- XGBoost for package volume prediction
- Genetic algorithms for vehicle routing
- Pareto optimization for multi-objective problems

## Requirements
- Python 3.8+
- NumPy
- Pandas
- Scikit-learn
- XGBoost
- Matplotlib
- DEAP (for genetic algorithms)

## Outputs
- Prediction accuracy reports
- Vehicle scheduling plans
- Cost optimization analysis
- Sensitivity analysis results
